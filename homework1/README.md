# Визуализация динамической функции с изолиниями

## Описание

Программа визуализирует функцию

[
f(x, y, t)
]

на прямоугольной области, разбитой на сетку. На каждом кадре вычисляются значения функции в вершинах сетки, строится цветовая карта поверхности и набор изолиний.

Для построения изолиний используется алгоритм **Marching Squares**.

В качестве демонстрационной функции используется:

[
f(x,y,t)=\sin(x+t)\cos(y-t)
]

что создает анимированную волновую поверхность.

## Сборка и запуск

```bash
mkdir build
cd build/
cmake -DCMAKE_POLICY_VERSION_MINIMUM=3.5 ..
make -j4
./homework1
```

## Управление

* `SDLK_LEFT` — уменьшение детализации сетки
* `SDLK_RIGHT` — увеличение детализации сетки
* `SDLK_DOWN` — уменьшение количества изолиний
* `SDLK_UP` — увеличение количества изолиний

## Возможности

* динамическое обновление функции каждый кадр;
* цветовая визуализация значений функции;
* построение изолиний в реальном времени;
* изменение детализации сетки;
* изменение количества уровней изолиний.
